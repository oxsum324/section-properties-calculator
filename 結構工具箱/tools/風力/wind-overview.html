主結構工具

主結構抗風骨架 (MWFRS) 計算路線；總高與兩向迎風寬決定整體風力。
① 矩形建物 MWFRS 第二章 主流程 逐層分析、X / Y 兩向、橫風向、扭矩；柔性建物完整 Gf。

外牆與屋面構材 (C&C)

構材與包覆物採區域 GCp 設計，外牆 4/5、屋面 1/2/3。
② 區域風壓 (C&C) 第三章 §3.3 / §3.5 外牆與屋面區域逐區設計風壓；正/負壓組合，主控標示自動高亮。 ③ 女兒牆風壓 第三章 §3.4 / 圖 3.4-3.5 MWFRS 與 C&C 雙模式；建築物屋頂 / 單一屋頂女兒牆全條件展開。 ④ 開放式建築屋面 第三章 §3.4 / 圖 3.3 單斜 / 雙斜開放屋頂之 Cpn 淨風壓係數。

特殊標的物

附屬構造物之風力骨架；輸出 F、V、M 與分段剪力表。
⑤ 實體標的物 第二章 表 2.13 實體箱體、看板、設備罩體；單一作用點或分段。 ⑥ 透空 / 格子式 第二章 表 2.11 / 2.15 中空標示物、格子構架、桁架高塔。 ⑦ 煙囪 / 高塔 / 水塔 第二章 表 2.12 塔狀構造物 + 塔頂附加風力。 ⑧ 圍籬牆 / 獨立招牌 第二章 表 2.10 座地 / 架高雙模式；形心高度速度壓 + F/V/M。 ⑨ 形狀 / 修正係數 第二章 表 2.10/2.13/2.14 Cf 與修正係數查表，供回填至其他工具。 ⑩ 特殊修正 / 折減 附加流程 遮蔽折減、透氣外牆折減等 λ 係數整理。

常用工作流程

建議的耐風檢核順序（依案件規模可選擇性執行）：
  1. 主結構檢核：輸入基本參數後執行 ① 矩形建物 MWFRS，得到逐層風力、底部剪力、彎矩。
  2. 包覆物檢核：以同一案件參數執行 ② C&C，逐區域查最大正/負壓。
  3. 女兒牆 / 突出物：若有女兒牆執行 ③；若有附屬看板/設備執行 ⑤–⑧。
  4. 形狀修正：特殊形狀（角隅圓弧、傾斜面）以 ⑨ 查係數，回填至 ⑤–⑧。
  5. 特殊折減：遮蔽、透氣外牆、風洞修正等以 ⑩ 整理依據。
