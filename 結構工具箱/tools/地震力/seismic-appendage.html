本頁定位與適用範圍
本頁專注第四章附屬構造物之局部計算書，適合快速計算附屬構件之水平與垂直地震力，並直接整理成章節型報表。
適合：機電設備、管線、吊掛設備、外掛構件、非結構牆、天花、招牌等附屬構造物。
不適合：完整建築物通盤檢討、主結構等值靜力法、第五章雜項工作物案件。
若需完整主結構耐震整理，請回 等值靜力分析。
版本與更新重點
本頁目前版本為 V2.5，定位為第四章附屬構造物的獨立局部工具，支援獨立輸入、沿用等值靜力摘要與案件 JSON 回讀三種模式。
本版已完成：表 4-1 / 表 4-2 型式整理、式(4-1)~式(4-3) 計算、Rpa 盆地調整、垂直力近斷層比值、地區回填、案件 JSON 匯入匯出、詳算式 / 簡易結果列印模式、構材類型分頁、輸入檢核與示意圖。
建議用法：作為第四章專章計算與送審前整理頁；若需與主結構耐震結果一併統整，請回 等值靜力分析。
