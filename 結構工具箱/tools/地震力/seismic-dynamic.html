判讀建議

    本頁定位與適用範圍
    本頁是「動力分析結果摘要與靜力比對」工具，用來承接外部模型分析成果，整理設計譜參數、動力基底剪力與動 / 靜比。
    適合：ETABS / SAP2000 / MIDAS 等反應譜分析結果之案件整理、送審前比對、報表化輸出。
    不適合：直接建立模型、求模態、執行反應譜分析或取代分析軟體本身。
    若只需完整靜力法計算，請回 等值靜力分析。
    版本與更新重點
    目前頁面版本為 V1，定位為動力分析成果的整理、比較與列印，而不是分析求解器本身。
    本版已完成：靜力摘要載入、案件資料銜接、DBE / MCE 反應譜繪製、Tx / Ty 設計週期標示、動 / 靜比整理與列印。
    建議用法：先於模型軟體完成分析，再把 X / Y 向基底剪力與週期帶入本頁，作為報表化整理與比對。
    若需要正式靜力法主算，請回 等值靜力分析。
