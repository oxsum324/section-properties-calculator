開挖擋土支撐計算工具 本機服務工具

FastAPI + React + SQLite｜支撐構件檢核與報表輸出

工具狀態

目前定位 已納入工具箱索引，但不是純靜態 HTML；需啟動本機後端服務後使用。
主要流程 匯入 .RIO / .LST / .o 分析文字檔，整理支撐、橫擋、斜撐、大角撐與柱構件檢核，並輸出報表。
目前限制 柱構件檢核仍屬第一版移植；複雜案例應與既有 Excel / 計算書交叉比對。

啟動方式

cd C:\Users\USER\Desktop\AI\小工具製作\開挖擋土支撐 .\start_html_mode.ps1
開啟本機服務 查看 README 返回結構工具箱
若要正式納入靜態部署，需另外處理前端 base path、API 服務部署與資料檔儲存邊界；目前此入口以本機工程工作流為準。
