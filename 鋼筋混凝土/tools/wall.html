面外載重 (Mua / Pu,oop)

適用條件: kℓc/h ≤ 30, Pu ≤ 0.06 fc'·Ag, ρℓ ≥ 0.01。Mua 為含 P-δ 二階放大後的彎矩需求 (tf·m/m)。

三角形土壓 + 矩形超載 + (可選) 三角形水壓；可選懸臂牆或簡支條帶估算模型。系統以所選模型計算彎矩後乘 1.6 因素，並自動覆蓋上方 Mua 顯示。此功能屬概算，正式設計仍應以實際受力模型為準。

耐震載重 ＆ 邊界構材參數 SSW

用於檢核是否需要特殊邊界構材 (>0.2fc')。僅在勾選耐震模式或結構牆時適用。

邊界構材詳細 (若需設置)

依規範 18.10.6.4 — 若 c ≥ ℓw/(600·(1.5·δu/hw)) 須設置邊界構材, 並沿高度延伸 max(ℓw, Mu/(4Vu))。

輸入同層各片牆段，檢核 ΣVn ≤ 2.12·λ·√fc'·ΣAcv (規範 18.10.4.4) 並警示力集中。
