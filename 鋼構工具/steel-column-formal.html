單位系統

柱正式頁可在舊制與 SI 輸入間切換；斷面尺寸與規格選單固定維持 mm。

輸入：舊制
斷面尺寸 mm H / B / tw / tf / R / B / D / tnom
長度與穩定參數 cm Lx / Ly / Lbx
荷重與強度 tf / tf·m Pu、Mux、Muy、設計強度與可用容量
材料應力 kgf/cm² Fy、Fe、Fcr
SI 常用換算

1 tf = 9.80665 kN｜1 tf·m = 9.80665 kN·m｜1 kgf/cm² = 0.0980665 MPa｜1 cm = 10 mm

輸入欄位、摘要卡與匯出報表會跟著切換；斷面尺寸欄位與規格下拉固定維持 mm。

計算流程與符號詞典仍沿用核心計算單位 `kgf/cm² / tf / cm` 顯示代入。
