# 結構計算小工具

本專案提供結構計算、報告整理與交付前檢核的小工具。語彙重點是區分可交付的計算附件、工具頁面提示與內部驗證證據。

## Language

**計算附件**:
提供本公司整理報告書時使用的結構計算支援資料，應聚焦於計算依據、輸入、結果、採用條件與必要覆核提醒。
_Avoid_: 工具狀態總覽、首頁狀態快照

**計算書**:
可列印或匯出的正式計算記錄，內容包含規範判定、採用參數、計算過程、結果與交付所需附註。
_Avoid_: 工具閱讀狀態、內部稽核狀態

**報告閱讀狀態**:
工具頁面用來提示報告可讀性、缺漏與優先審閱方向的狀態；它不是計算結論，也不是交付附件的一部分。
_Avoid_: 計算附件、計算書章節

**頁面專用閱讀狀態**:
明確只允許呈現在頁面或工具本身的報告閱讀狀態，列印與 PDF 匯出時應排除。
_Avoid_: PDF 附件、列印內容、計算書附錄

**首頁狀態快照**:
工具首頁呈現目前驗證與閱讀狀態的摘要，供使用者判斷哪些工具可優先檢視或仍需補強。
_Avoid_: 計算書摘要、交付封面

**發布佐證**:
用來證明工具目前狀態的 preflight、稽核或 smoke 測試結果；它支撐工具可信度，但不是工程判定本身。
_Avoid_: 規範判定、設計結論
